# Mobile Module 00

Learn the basics to start building a simple mobile application.
I haven chosen to code in React Native with Expo.
I used Expo Go on my phone to open the application. 

## Get started

1. Install Node.js

  It will allows you to have the commands npm and npx.
  You can visit this website [NodeJs.org](https://nodejs.org/fr)

2. Move on every exercices and launch project

   ```bash
   npm install
   npx expo start
   ```
3. Open the app

   Scan the QR code with your phone.
